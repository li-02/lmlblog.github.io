只复制不存在的属性

function merge(target, ...sources) {
    for (let source of sources) {
        for (let key of Object.keys(source)) {
            if (!(key in target)) {
                target[key] = source[key];
            }
        }
    }
    return target;
}
console.log(merge({ x: 1 }, { x: 2, y: 3 }))
// { x: 1, y: 3 }

# 11.1.1 Set类

集合是一组值,与数组类似,但是集合没有索引或顺序,不允许有重复值

使用Set()构造函数创建集合对象:

let s=new Set(); // 空集合
let c=new Set([1,2,c]);

Set()构造函数的参数不一定是数组,可迭代对象就行

let b=new Set(s); // 复制s元素的新集合
let unique=new Set("miss"); // 三个元素,m i s

集合的size属性表示集合的长度:

unique.size; // 3

集合可以使用add(),delete(),clear()添加或删除元素

let s = new Set();
s.size; // 0
s.add(1);
s.size; // 1
s.add(true);
s.size; // 2

s.add([1, 2]); // 添加一个数组
s.size; // 3
console.log(s);

console.log(s.delete([1, 2])); // false,因为[1,2]和[1,2]是两个对象的引用
// 要想删除可以这样
let arr = [1, 2];
s.add(arr);
s.delete(arr);

s.clear(); // 清空数组

has( )检查某个值是不是集合的成员

let set = new Set([1, 2, 3]);
set.has(1); //true
set.has("1"); //false

Set类可迭代

for (let s of set) {
    console.log(s);
}

可以使用扩展操作符 ... 将集合转化为数组或参数

console.log([...set]);
console.log(Math.max(...set));

Set是无序的,但是Set会记录插入的顺序,第一个元素第一个迭代,最后添加的最后迭代

除了迭代外,还实现了forEach()方法

let t = 2;
set.forEach(n => {
    t += n;
})
console.log(t);

# 11.2 定型数组和二进制数据

定型数组和常规数组的重要区别:

  • 定型数组的元素全部都是数值,定型数组允许指定存储在数组中的数值的类型
  • 创建定型数组时必须指定长度,且长度不能再改变
  • 定型数组的元素在创建时始终都会被初始化为0
